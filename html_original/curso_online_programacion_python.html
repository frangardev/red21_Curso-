Curso Online de Programación Python 2025

Habilitado por INEFOP

Python es el lenguaje de programación de más rápido crecimiento y mayor demanda laboral, siendo utilizado por programadores y analistas de datos en áreas como inteligencia artificial y innovación en empresas. Python ha revolucionado la programación por varios motivos entre los que se destacan:

  • Escritura de código fácil y muy simplificada. Siendo extremadamente flexible y fácil de aprender en comparación con otros lenguajes..
  • Portable : puede funcionar muy bien en cualquier sistema operativo: Windows, Mac o Linux.
  • Con este lenguaje sencillo de aprender se han creado aplicaciones como Instagram, Dropbox, Pinterest y buena parte de las funciones de Google y Facebook.
  • Es el lenguaje más utilizado en el análisis de datos, Big data y Inteligencia artificial a través de Deep Learning y Aprendizaje automático.

Objetivos

  • Aprenderás desde cero los fundamentos de Python que te permitirán convertirte en un Programador profesional.
  • Podrás crear programas específicos para la demanda laboral actual.
  • Utilizar Python para realizar análisis de datos sobre bases de datos masivas (big data) y un interpretador inteligente de esos datos (aprendizaje automático) .

A quién está dirigido

Todo público. Podrás tomar el curso con cualquier tipo de computadora, tablet o smarthphone.

Temario

Módulo 1
  • Introducción. Qué aprenderás. Cómo organizar el tiempo de aprendizaje.
  • Python y otros lenguajes. Entornos de programación. Como utilizar Jupyter Notebook.
  • Escribiendo el primer código. Las variables. Tipos de variables.
  • Operaciones básicas. Introducción a estructuras de control.
  • Cómo instalar Python en una computadora.
  • Introducción a la Programación orientada a objetos.
  • Trabajo práctico sobre cuaderno interactivo 1.
Módulo 2
  • Tipos de operadores.
  • Estructuras de control de flujo de decisiones.
  • Funciones. Funciones integradas en Python. Cómo crear funciones.
  • Cómo escribir código ordenado.
  • Formatos de números.
  • Paquetes y módulos. Cómo instalar paquetes.
  • Cómo resolver un problema con seudocódigo.
  • Trabajo práctico sobre cuaderno interactivo 2.
Módulo 3
  • Estructuras de datos en Python: textos, listas, tuplas, diccionarios y conjuntos.
  • Cómo crear y editar listas.
  • Cómo crear y editar diccionarios. Ejemplos.
  • Listas y diccionarios de datos. Como acceder a los datos incluidos en una lista o diccionario de datos.
  • Conjuntos de datos o datasets.
  • Trabajo práctico sobre cuaderno interactivo 3.
Módulo 4
  • Acceso a bases de datos desde Python. Trabajando con el paquete Pandas.
  • Cómo crear un marco de datos. Selección de datos.
  • Creación de máscaras booleanas. Uso de condiciones lógicas.
  • Cómo agrupar y resumir datos.
  • Importación de datos. Trabajando con datos externos.
  • Creación de gráficos personalizados.
  • Analisís de datos. Filtrado de valores atipicos. Representación gráfica.
  • Introducción al aprendizaje automático.
  • Trabajo práctico sobre cuaderno interactivo 4.
  • Proyecto final: Creación de un cuaderno Jupyter sobre todo lo aprendido en el curso.

Cómo funciona

  • El curso es a través de Internet en la modalidad de curso online.
  • El contenido se configura de acuerdo a los conocimientos previos que tengas, así como al tipo de computadora, tablet o teléfono móvil que utilices.
  • Las clases y los ejercicios se cursan desde nuestra aula virtual desde cual tendrás acceso a las lecciones y entorno de programación en tiempo real.
  • No necesitas estar conectado a una determinada hora. Puedes tomar las lecciones en cualquier horario y todos los días de la semana (24 hs).
  • Aprenderás a tu ritmo. Para culminar el curso tienes un plazo de un año desde la fecha de comienzo.
  • Puedes consultar al tutor todos los días de lunes a domingo de 11 a 21 horas.

Utilización de entorno de programación Jupyter. Verificas el código que escribes en tiempo real.
